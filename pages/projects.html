LGBM Tanks

Overview

I was reminded a retro game Scorched Earth. Why not code it in Python?

I've first steps in pygame and coded a simple game where two players shoot to themselves with tanks.

I've implemented as bullet shot engine with air conditions (e.g. wind). It's based on physical model of movement

Why not use AI to control those tanks?

I've run an following experiment

  • forget about physical movement model - you know nothing about physics
  • however, you are able to train and implement machine learning model
  • train LGBModels to predict a result of a shot basing on state (e.g. player coordinates, wind) and action (shot angle, shot power)
  • choose action that leads to the best result

AI Tanks started to shoot precisely (~) after 1600 games

Tech Stack

  • python 3.12
  • pygame
  • LightGBM

Deforestation Recognition

Overview

I was invited to the hackathon on Linkedin by Schneider electric recruiters.

Dataset contained satellite photos of 3 deforestation types.

The task was to train classification model to recognize the deforestation type.

The main problem was imbalanced data

  • I splitted datasets in 2 parts with balanced classes (split by photo year)
  • trained two models for different dataset
  • multiplicated datasets with synthetic photos

Tech Stack

  • python
  • tensorflow
  • cv2
